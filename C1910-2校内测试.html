2 下午

考得心态爆炸，题目是一道除法分块：

F(n) = (n % 1) + (n % 2) + (n % 3) + ...... (n % n)。其中%表示 Mod， 也就是余数。 例如 F(6) = 6 % 1 + 6 % 2 + 6 % 3 + 6 % 4 + 6 % 5 + 6 % 6 = 0 + 0 + 0 + 2 + 1 + 0 = 3。 给出 n，计算 F(n), 由于结果很大，输出 Mod 1000000007 的结果即可。
